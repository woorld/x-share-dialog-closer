# X Share Dialog Closer

## なにこれ

Xの共有ボタンを押したときに開くタブに表示されるダイアログを自動で閉じます。
閉じた後は通常の「ポストする」ボタンから投稿を行えます。

## 使い方

1. このページのどっかからソースをZIPでダウンロード
2. 展開
3. `chrome://extensions`にアクセス
4. 「パッケージ化されていない拡張機能を読み込む」を押す
5. 展開したフォルダ（`content.js`とかがあるフォルダ）を選択

## モチベーション

- 共有ボタンから投稿するとき、投降後にタブを閉じようとすると「本当に閉じますか？」みたいな確認が出るのが鬱陶しかった
  - ダイアログから投稿すると元々の投稿欄にテキストが残ってしまい確認が出る
  - ならいっそのことダイアログを閉じて元々ある投稿ボタンを使えば解決じゃん？ということで作成
